# 📝 欢迎使用 Markdown KB

这是一个基于 Tauri + Vue 3 + FastAPI 构建的本地 Markdown 笔记与知识库应用。

## 主要功能

### 🔗 双向链接 (Wiki Links)

使用 `[[笔记名称]]` 语法创建双向链接：

- 链接到 [[编程技巧]]
- 链接到 [[阅读笔记]]
- 链接到 [[项目笔记]]

当你在一个笔记中引用另一个笔记时，被引用的笔记会自动记录"反向链接"。

### 🕸️ 知识图谱

点击顶部菜单栏的"🕸️"图标，可以查看所有笔记之间的关联关系图谱。

### 🔍 全文搜索

使用 `Ctrl+K` 快捷键或顶部搜索栏，可以快速搜索所有笔记内容。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 聚焦搜索框 |
| `Ctrl+N` | 新建笔记 |
| `Ctrl+S` | 保存当前笔记 |
| `Esc` | 关闭对话框 |

## 开始使用

1. 点击左侧文件树的 "+" 按钮创建新笔记
2. 在编辑器中使用 Markdown 语法编写内容
3. 使用 `[[笔记名]]` 链接到其他笔记
4. 点击预览中的链接可以快速跳转到对应的笔记

---

> 💡 提示：所有笔记都保存在本地文件系统中，你可以随时用其他 Markdown 编辑器打开它们。
