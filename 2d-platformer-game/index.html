🎮 2D 平台跳跃游戏

经典横版跳跃游戏 - 收集金币，躲避危险！

📖 游戏说明

🎯 游戏目标

控制角色在平台间跳跃，收集尽可能多的金币，同时躲避危险物，挑战更高分数！

🎮 操作方式

  • ← → 方向键：左右移动
  • ↑ 方向键 / 空格：跳跃
  • 空中再按跳跃键：二段跳
  • S：保存游戏进度
  • ESC：暂停游戏

⭐ 得分规则

  • 收集 金币：每个 +1 分
  • 完成关卡：获得额外奖励
  • 游戏结束时统计最终得分

💀 生命值系统

游戏开始时有 3 条生命

  • 触碰尖刺：失去 1 条生命
  • 掉落屏幕：失去 1 条生命
  • 生命耗尽：游戏结束

💾 存档功能

按 S 键可随时保存游戏进度

  • 保存当前关卡
  • 保存已收集金币数
  • 保存剩余生命值
  • 保存玩家位置

💡 小提示

  • 善用二段跳跨越更远距离
  • 注意观察尖刺位置，提前跳跃
  • 定期保存游戏进度
  • 金币可以在平台上方找到
