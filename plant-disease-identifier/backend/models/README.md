# 模型文件目录

## 说明

此目录用于存放训练好的植物病害识别模型文件。

## 支持的模型格式

1. **PyTorch 模型格式 (.pt / .pth)**
   - 文件路径: `./models/plant_disease_model.pt`
   - 使用方法: 直接将训练好的模型文件放入此目录即可

2. **ONNX 模型格式 (.onnx)**
   - 文件路径: `./models/plant_disease_model.onnx`
   - 需要修改 `app/core/config.py` 中的 `MODEL_FORMAT` 为 `"onnx"`

## 模型要求

- 输入图片尺寸: 224x224 像素
- 输入通道: 3 (RGB)
- 归一化参数:
  - mean: [0.485, 0.456, 0.406]
  - std: [0.229, 0.224, 0.225]
- 输出类别数: 14 类 (详见 `app/core/config.py` 中的 `DISEASE_CLASSES`)

## 类别列表

1. 健康
2. 番茄早疫病
3. 番茄晚疫病
4. 番茄叶霉病
5. 番茄灰霉病
6. 黄瓜霜霉病
7. 黄瓜白粉病
8. 黄瓜炭疽病
9. 苹果腐烂病
10. 苹果炭疽病
11. 苹果轮纹病
12. 葡萄白粉病
13. 葡萄霜霉病
14. 葡萄炭疽病

## 快速开始

1. 将训练好的模型文件放入此目录
2. 确保模型文件名与 `app/core/config.py` 中的配置一致
3. 运行后端服务，模型会在启动时自动加载

## 注意事项

- 如果没有训练好的模型，系统会使用未经训练的 ResNet50 模型进行测试
- 实际使用时请替换为训练好的模型文件以获得准确的识别结果
