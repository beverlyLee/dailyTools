# 20个图像处理工程项目列表

这份列表包含20个图像处理工程项目，专注于解决生活、学习和工作中的具体痛点。所有项目均基于 **HTML5 Canvas + Vanilla JavaScript (ES Modules)** 技术栈，确保环境统一且无需复杂构建工具，非常适合 Trae Solo 的快速生成模式。

## 1. 纸质文档阴影去除工具
*   **项目名称：** `doc-shadow-remover`
*   **项目业务领域：** 办公效率 / Web前端
*   **任务类型：** 0-1代码生成
*   **Prompt:**
    *   **需求:** 需求是开发一个纸质文档阴影去除工具，在 `/doc-shadow-remover/` 文件夹下，主要实现的是解决手机拍摄纸质文档时，手指或光源造成的阴影导致文字难以辨认的问题。
    *   **技术栈:** HTML5, CSS3, JavaScript (ES Modules), Canvas API.
    *   **环境配置:** 纯静态文件，无需构建工具。推荐使用 VS Code 的 Live Server 插件启动本地服务。
    *   **功能模块拆分:**
        1.  **阴影掩码生成 (`ShadowMask.js`):** 将图像转换到 HSV 色彩空间，通过分析 Value（明度）通道的局部极小值，结合形态学膨胀操作，生成精确的阴影区域软掩码。
        2.  **自适应补偿 (`BrightnessCompensator.js`):** 仅对掩码区域进行亮度提升，保持色调和饱和度不变，避免纸张发黄或偏色。
        3.  **文字锐化 (`TextSharpen.js`):** 去阴影后，对文字边缘进行轻微的非锐化掩蔽(Unsharp Mask)，提升 OCR 识别率和阅读清晰度。
*   **项目验证方法:** 用手机拍摄一张被手指严重遮挡阴影的文档，上传处理后，检查阴影区域是否变白且下方文字清晰可见。
