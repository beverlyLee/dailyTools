📌 如何获取 API Key？
1. 登录火山引擎控制台：https://console.volcengine.com
2. 进入「智能方舟（Ark）」服务
3. 在左侧菜单选择「API Key 管理」
4. 点击「新建 API Key」创建并复制
📌 如何获取模型名称？
模型名称格式示例：
doubao-seed-2-0-code-preview-260215

获取步骤：
1. 在火山引擎控制台进入「智能方舟（Ark）」
2. 进入「模型广场」或「模型推理」
3. 选择需要的模型（如豆包、GPT 等）
4. 复制模型名称填入上方输入框

API 调用示例：
{
  "model": "doubao-seed-2-0-code-preview-260215",
  "messages": [
    {"role": "user", "content": "Hello"}
  ]
}
📌 如何使用本地代理？
启动代理服务器：
1. 打开终端，进入项目目录
2. 运行命令：node proxy-server.js
3. 看到 "代理服务器已启动" 说明成功
4. 在上方勾选"使用本地代理"选项
5. 点击"测试连接"验证

代理服务器作用：
- 绕过浏览器 CORS 限制
- 转发请求到火山引擎 API
- 安全处理 API Key
📌 常见问题
Failed to fetch 错误：
这通常是浏览器 CORS 限制导致的。解决方案：
1. ✅ 推荐：启用"使用本地代理"并运行 node proxy-server.js
2. 或使用浏览器插件（如 Allow CORS）临时绕过
3. 确认网络连接正常

401 Unauthorized：
API Key 无效或已过期，请重新生成 API Key

404 Not Found：
Model ID 不正确或端点已删除
